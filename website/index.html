设计与技术

重塑私人影音播放的每个细节

结合高性能底层开发与精美的前端界面，RPlayer 解决了云端播放卡顿、数据混乱、升级繁琐的核心痛点。

极速 Rust 编译核心

播放器底层数据流与SQLite处理全部由高性能 Rust core 驱动，支持路径自动规范化，合并并消除重复的冗余文件记录，保证几十毫秒内的加载反应与极低的内存开销。

WebDAV 流媒体流式同步

支持对接各大网盘及 NAS 自建 WebDAV 协议。智能剥离不一致路径，确保进度断点记录、横竖屏偏好等配置跟文件严格挂载至唯一的目录ID，即使断线重连，体验依然连贯。

智能 TMDB 元数据刮削

全自动多线程匹配后台，不仅能刮削电影及电视剧名称、评分、演职员，还扩展了针对“季（Season）”维度详情字段的精准记录，包含季海报、简介及总集数，搭建出属于您的绝美媒体库。

内置无感版本热更新

客户端集成新版自动升级服务，在每次启动时都会静默比对您的最新 GitHub Release，一旦发布新版，立刻提供清晰的更新日志与一键安全更新下载，始终使用最新体验。

v1.0.0 2026-06-12

首个正式版本发布

核心功能全面对齐，通过 GitHub Actions 工作流实现了 Rust Core (arm64/v7) 与 Flutter APK 的全自动化编译与版本打包挂载。

  • 路径规范化合并：彻底解决由于 WebDAV 路径前缀（如 `/dav`）不一致引起的多个重复目录及对应的文件状态散落问题，SQLite 自动执行物理及逻辑文件夹合并更新。
  • 季维度数据支持：打通 TMDB 季维度元数据详情存取（包含季海报图、集数、简介等），完善客户端的多季展示结构。
  • 数据库状态瘦身：重构并废弃明文保存 WebDAV 密码的大 JSON 存储，播放进度和偏好设置全面改由分表管理，大幅提升数据检索安全性与读写性能。
  • 内置升级检测：在 `PlayerShell` 顶层与个人中心页集成一键版本升级通知与 APK 静默拉起下载，无需应用商店即可完成更新。
