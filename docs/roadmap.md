# 开发路线

## M1: 可扫描的媒体库

- 建立 Flutter 基础页面
- Rust 文件扫描
- 影片名解析和候选生成
- SQLite 媒体库
- 手动刷新和手动匹配入口

## M2: TMDB 海报墙

- TMDB 搜索和详情
- 海报/背景图缓存
- 电影/剧集详情页
- 匹配置信度展示

## M3: libmpv 本地播放

- 桌面端 libmpv 初始化
- 打开本地视频
- 播放、暂停、seek、音量、倍速
- 播放进度保存

## M4: 字幕

- 同目录字幕发现
- 字幕轨道切换
- 字幕延迟
- 字幕样式设置

## M5: WebDAV

- WebDAV 源配置
- 远程目录浏览
- 远程视频播放
- Range 支持检测

## M6: 弹幕

- danmu_api 配置
- 自动匹配弹幕
- 弹幕加载、缓存
- Flutter 弹幕渲染层
- 弹幕过滤、透明度、速度、密度、时间偏移

## M7: 移动端与发布

- Android/iOS libmpv 打包
- 移动端权限和后台音频策略
- CI 构建
- 自动更新策略
